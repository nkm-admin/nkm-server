# nkm后台管理系统初始化模版-后端

> 后端使用Nodejs + Koa2 + Mysql + Redis完成

## 使用前提

* 本地需要`mysql`与`redis`环境

* 克隆代码到本地
  `git@github.com:nkm-admin/nkm-server.git`

* 导入mysql
  将项目根目录的`nkm_admin.sql`导入至数据库，admin账号默认密码：123456

## 安装项目依赖
```
yarn
or
npm install
```

### 启动项目

> 项目默认端口为：3333

```
yarn start
or
npm run start
```
